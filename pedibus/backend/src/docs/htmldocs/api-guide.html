Gestione prenotazioni

Aggiunta prenotazione

POST /reservations/{id_linea}/{data} @RequestBody ReservationResource reservationResource Aggiunge una prenotazione e ne restituisce l’identificatore univoco

Example Request

POST /reservations/linea1/2020-02-14/ HTTP/1.1
Content-Length: 77
Authorization: Bearer eyJhbGciOiJIUzI1NiJ9.eyJzdWIiOiJ0ZXN0Lkdlbml0b3JlQHRlc3QuaXQiLCJyb2xlcyI6WyJST0xFX1VTRVIiXSwiaWF0IjoxNTgxNTE2MzY4LCJleHAiOjE1ODE1MjcxNjh9.oWTbv3q9HVunowjagHBzA7FhKkZW8bkkTv7kqp5M6A8
Content-Type: application/json
Host: localhost:8080

{
  "cfChild" : "RSSMRA30A01H501I",
  "idFermata" : 1,
  "verso" : true
}

CURL request

$ curl 'http://localhost:8080/reservations/linea1/2020-02-14/' -i -X POST \
    -H 'Authorization: Bearer eyJhbGciOiJIUzI1NiJ9.eyJzdWIiOiJ0ZXN0Lkdlbml0b3JlQHRlc3QuaXQiLCJyb2xlcyI6WyJST0xFX1VTRVIiXSwiaWF0IjoxNTgxNTE2MzY4LCJleHAiOjE1ODE1MjcxNjh9.oWTbv3q9HVunowjagHBzA7FhKkZW8bkkTv7kqp5M6A8' \
    -H 'Content-Type: application/json' \
    -d '{
  "cfChild" : "RSSMRA30A01H501I",
  "idFermata" : 1,
  "verso" : true
}'
